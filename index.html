ABOUT THIS DASHBOARD

This interactive visualization explores the lyrical and musical patterns of 65 major college fight songs. Click any node or logo to analyze that school's fight song, or hover over a trope to see which songs utilize it.

  • TROPE-O-METER: Tracks which specific clichés show up in fight songs. Click any logo to highlight that school's song, or hover over a trope icon to view a description of that trope and which songs utilize it.
  • JACCARD NETWORK: A network graph connecting songs whose lyrics share at least 75% similarity in tropes calculated by Jaccard index. Mousing over or clicking a node highlights its "neighbors." The color of a node represents the number of tropes in the song.
  • SONG TOPOLOGY: A scatter plot of tempo (BPM) vs. duration (sec) with added densities. The color of a node indicates how many times the word "fight" is sung.
  • ORIGIN STORY: An UpSet plot showing how songs were created based off the power set of {Student Author, Contest Winner, Official Song}.
