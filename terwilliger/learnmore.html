Finding a Basis for T(x)

Mathematical justification of the algorithm
Barnabás Valkó  ·  University of Wisconsin–Madison

The Terwilliger algebra T(x) of a graph is a finite-dimensional associative matrix algebra that encodes the structural symmetry of the graph relative to a fixed base vertex x. It is generated by the adjacency matrix A and a family of diagonal projection matrices called dual idempotents. The algorithm computes a canonical basis for T(x) in three stages.

1
Generators: the adjacency matrix and dual idempotents

Let Γ = (V, E) be a finite connected graph with n vertices. Fix a base vertex x ∈ V. Breadth-first search from x computes the exact shortest-path distance ∂(x, v) to every other vertex in O(n + |E|) time. The diameter from x is D = max ∂(x, v).

Definition — Dual Idempotents
For 0 ≤ i ≤ D, the dual idempotent E*i is the diagonal n×n matrix whose (j,j)-entry is 1 if ∂(x, vj) = i, and 0 otherwise. These satisfy E*i E*j = δij E*i (mutual orthogonality) and ∑ E*i = In (partition of identity).

The Terwilliger algebra is then defined as the subalgebra of Matn(ℝ) generated by these matrices:

T(x) = spanℝ{ w(A, E*0, …, E*D) : w is any noncommutative word in the generators }
Generator A
Adjacency matrix
Standard 0/1 symmetric matrix. Encodes the graph's edge structure globally.
Generators E*0, …, E*D
Dual idempotents
Diagonal projections onto distance layers. Encode local structure relative to x.
2
Gram–Schmidt word enumeration

Since T(x) ⊆ Matn(ℝ) and dim Matn(ℝ) = n², the algebra is at most n²-dimensional. The algorithm constructs a basis by a BFS over all noncommutative words in the generators, testing each for linear independence via Gram–Schmidt orthogonalization.

Key Lemma — Gram–Schmidt Test
Flatten each n×n matrix M to a vector v ∈ ℝn². Compute the residual r = v − ∑ ⟨v, b̂j⟩ b̂j over the current orthonormal basis. Then M is linearly independent of the current basis if and only if ‖r‖ > ε.

The BFS seeds the basis with E*0, …, E*D and then iterates: each newly added basis matrix is multiplied by every generator, and the result is tested for linear independence. Only matrices that survive the test are added to the frontier for the next round.

Theorem — Completeness
When the BFS frontier is empty (no new independent matrices found in a generation), the current span is closed under right-multiplication by all generators. Since the algebra is also closed under left-multiplication and contains all generators, this span equals T(x). Termination is guaranteed because the basis size is bounded by n².

In practice the algebra closes at dimension far smaller than n². For the Foster graph (90 vertices) the dimension is 257; for the Biggs-Smith graph (102 vertices) it is 451.

3
RREF canonicalization

Once a linearly independent spanning set {M1, …, Md} is found, it is already a basis. The algorithm applies the Reduced Row Echelon Form (RREF) to the augmented matrix formed by stacking the flattened basis matrices as rows alongside a d×d identity block:

[ m1 | · · · | md ]T | Id

Row operations preserve the row space, so the RREF produces a new basis spanning the same algebra. Because the stacked matrix has full row rank d, the RREF yields exactly d nonzero rows, each with a unique pivot position among the n² matrix entries. The augmented identity block tracks the change-of-basis transformation, expressing each canonical basis element as a linear combination of the original word matrices.

Proposition — Uniqueness
The RREF of a matrix is unique (for a fixed column ordering). Therefore this procedure produces a canonical basis for T(x) that depends only on the graph and the choice of base vertex, not on the order in which words were enumerated.

SymPy's exact rational arithmetic is used for the RREF step, avoiding floating-point rounding errors in the pivot selection. The Gram–Schmidt stage uses floating-point arithmetic with tolerance ε = 10−9, which is adequate for graphs up to several hundred vertices.

4
Cell partition and association scheme

Each canonical basis matrix Ci has a well-defined support: the set of pairs (r,c) where the (r,c)-entry is nonzero. The algorithm assigns a label (A, B, C, …) to each basis matrix and marks every nonzero cell in the n×n grid with that label. Together these cells form a partition of V×V.

Definition — Cell Partition
The cell partition is the partition of V × V into classes RA, RB, … where (u, v) ∈ RL iff the (u,v)-entry of the canonical basis matrix with label L is nonzero. For distance-regular graphs this coincides with the association scheme defined by the graph.

The T-algebra structure constants with respect to this basis encode the combinatorial relationships between distance layers relative to the base vertex. The irreducible T-modules (ITMs) decompose the standard module ℝn into subspaces on which T(x) acts irreducibly, and their dimensions d1, d2, … satisfy:

dim T(x) = d1² + d2² + · · · + dk²

This sum-of-squares formula, visible in the table on the main page, is a consequence of the Artin–Wedderburn theorem applied to the semisimple algebra T(x).
